## GET /api/sat/regimenes-fiscales

Retrieves the catalog of fiscal regimes (Régimen Fiscal) from SAT. This catalog is used when creating or updating profiles and invoices to ensure compliance with Mexican tax regulations.

## Authentication

This endpoint requires authentication. Include your access token in the request cookies.

## Query Parameters

<ParamField query="tipo_persona" type="string" optional>
  Filter by person type. Options: `FISICA`, `MORAL`
</ParamField>

## Response

<ResponseField name="data" type="RegimenFiscalItem[]" required>
  Array of fiscal regime objects
  
  <Expandable title="Fiscal Regime Object">
    <ResponseField name="clave" type="string" required>
      Fiscal regime code (e.g., "601", "612")
    </ResponseField>
    
    <ResponseField name="descripcion" type="string" required>
      Description of the fiscal regime
    </ResponseField>
    
    <ResponseField name="aplica_persona_fisica" type="boolean" required>
      Whether this regime applies to individuals (Persona Física)
    </ResponseField>
    
    <ResponseField name="aplica_persona_moral" type="boolean" required>
      Whether this regime applies to legal entities (Persona Moral)
    </ResponseField>
  </Expandable>
</ResponseField>

## Example Request

```bash
curl -X GET "https://api.contafy.com/api/sat/regimenes-fiscales?tipo_persona=FISICA" \
  -H "Cookie: access_token=your_access_token"
```

## Common Fiscal Regimes

### For Persona Física (Individuals)
- **601**: General de Ley Personas Morales
- **605**: Sueldos y Salarios e Ingresos Asimilados a Salarios
- **606**: Arrendamiento
- **612**: Personas Físicas con Actividades Empresariales y Profesionales
- **621**: Incorporación Fiscal
- **626**: Régimen Simplificado de Confianza

### For Persona Moral (Legal Entities)
- **601**: General de Ley Personas Morales
- **603**: Personas Morales con Fines no Lucrativos
- **626**: Régimen Simplificado de Confianza

## Use Cases

- Populate regime dropdown when creating profiles
- Validate fiscal regime codes in invoices
- Filter available regimes based on person type
- Display regime information in user interfaces

<ResponseField name="400 Bad Request" type="error">

  ```json
  {
    "error": "Bad Request",
    "message": "Invalid tipo_persona. Must be FISICA or MORAL"
  }
  ```
</ResponseField>

<ResponseField name="500 Internal Server Error" type="error">

  ```json
  {

  }
  ```
</ResponseField>

<Note>
  This catalog is based on the official SAT c_RegimenFiscal catalog and is updated periodically to reflect changes in Mexican tax law.
</Note>

<Tip>
  When creating a profile, always validate that the selected fiscal regime is appropriate for the person type (física or moral).
</Tip>
